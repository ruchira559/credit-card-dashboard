# Credit Card Financial Dashboard

An interactive Power BI project designed to provide real-time insights into credit card customer demographics and transaction patterns.

## Project Overview
The goal of this project is to provide a 360-degree view of credit card operations. It consists of two main reports:
1. **Customer Report:** Focuses on demographics like age, income, and education.
2. **Transaction Report:** Analyzes spending habits, methods (Swipe/Chip/Online), and weekly revenue trends.

## Key Features
- **Dynamic Filtering:** Filter by Gender, Age Group, and Income levels across all pages.
- **Time Intelligence:** Track Week-on-Week (WoW) revenue growth.
- **Drillthrough:** Seamlessly move from a customer's profile to their specific transaction history.
- **Synced Slicers:** Maintain consistent views while switching between dashboards.

## Insights
* High-Value Segments: Customers with "Graduate" education and high income drive the highest revenue.
* Spending Channels: "Swipe" transactions remain the most frequent method for essential categories like Fuel and Grocery.
* Growth Trends: Revenue peaks were identified in specific quarters, allowing for better seasonal marketing planning.

## How to Use
1. Clone this repository.
2. Open the .pbix file.
3. If the data links are broken, point the data source to the CSV files.
